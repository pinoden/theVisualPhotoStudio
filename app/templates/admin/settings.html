Calendar Sync (iCal)

Subscribe to the studio calendar in Google Calendar, Apple Calendar, or Microsoft Outlook. Copy the URL below and add it as a calendar subscription.

How to subscribe:

  • Google Calendar: Settings → Add calendar → From URL → Paste the URL
  • Apple Calendar: File → New Calendar Subscription → Paste the URL
  • Outlook: Add calendar → Subscribe from web → Paste the URL

Studio Configuration

Edit the .env file in the project root to change:

  • BUNDLE_DISCOUNT_PCT — Bundle discount % (default: 10)
  • DEPOSIT_PCT — Deposit % of total (default: 30)
  • SLOT_INCREMENT_MINUTES — Time slot interval (default: 60)
  • ICAL_TOKEN — iCal feed security token
  • SQUARE_* — Square payment credentials
  • MAIL_* — Email settings

After editing .env, restart the Flask server for changes to take effect.
